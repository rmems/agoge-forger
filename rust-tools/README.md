# Agoge Rust Tools

Optional workspace for high-performance data processing, telemetry, and experimental framework integrations.

## Crates
- `agoge-cli`: Basic CLI wrapper.
- `agoge-jsonl`: JSONL validators.
- `agoge-gguf`: GGUF inspection placeholders.

## Future Plans (Commented out in Cargo.toml)
- `burn`, `candle`, `dfdx`: Rust ML frameworks.
- `perf-rs`: Telemetry.
- `axolotl-rs`, `aprender`, `rl`: Training abstractions.
